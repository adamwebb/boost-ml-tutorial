Logistic regression model

We can go ahead and build a simple logistic regression model with all these predictors.

We need to define a formula which takes looks something like this: response_variable ~ predictor_variables

The predictor_variables could be a list (e.g. Cell.size + Cell.shape + Mitoses), or we can use the wildcard . for all variables except the reponse variable.

Excercise: 1. Build and run a logistic regression model. Set data to BreastCancer and family to 'binomial' 2. Call summary() on our new model to get a summary of the model output

NoteHint 1

data = is our dataset family = should be binomial for logistic regression

TipFully worked solution: 
# where we will store our model
bc_model_logistic <- 
    glm(Class ~ .,   # our formula
            data = BreastCancer, 
            family = 'binomial')

# Take a look at the model summary:
summary(bc_model_logistic)

Visualising our model

Wouldn’t it be great if we could make some graphs which show predicted outcomes for different combinations of predictors?

It looks like Cl.thickness, Bl.cromatin and Bare.nuclei are pretty good predictors. We can use sJPlot again, but this time in 'pred' mode, which plots predictions.

So predictions are 0-100%. As malignancy is a binary output, we need to set a threshold to make a hard call between benign vs malignant. Where we set that will affect how sensitive and how precise our models are. This is the basis of the ROC curve and tells us how good our models are.

A few important things to note about our model:

  1. Some predictors don’t seem to be very good.

  2. It assumes that all predictors have a linear response. If we look at the categorical data before we converted it to numbers, we can see that sometimes the responses rise, peak and then fall again. Our logistic regression model misses this.

  3. Some variables correlate with each other. In a logistic regression, the effect is split across those variables and is diluted. We could be missing something important!

  4. If we look at our model output we see:

    (16 observations deleted due to missingness)

If any subject has any missing data, they get dropped from the analysis completely.

We will now look at how we can deal with 1, 2, 3 and 4.

  1. We could simplify our model by removing ineffective predictors - there is a fine balance between gaining information from extra predictors and overfitting (model too complex). Removing predictors can effect the model, especially if there are cryptic associations with other variables. We’ll look at ways to optimise models later.
  2. We could transform the data which shows a non-linear response.
  3. We could remove correlated variables, keeping the ones with most information
  4. Missing data can be a) ignored, and accept that we lose some data; b) treated by removing predictors with high levels of missingness; c) interpolated (e.g. replacing missings with mean values).
