Introduction

In this first section we will use the sample Wisconsin Breast Cancer dataset BreastCancer. This contains a number of histological variables, plus a response varable: is the cell benign or malignant? Using this dataset we will learn how to:

  • Load a sample dataset

  • Examine the contents of the dataset

  • Carry out a format conversion

  • Carry out a univariate analysis

  • Run a logistic regression

  • Build a table with the logistic regression

  • Plot the effect sizes of each predictor

  • Get a rough estimate of how accurate our logistic regression model is

You will need to run each code block in order or you’ll get some errors! You are welcome to chnage the code and experiment - everything here runs in your web browser. Some chunks may take some time to run.
